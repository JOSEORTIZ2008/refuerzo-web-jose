# Mantenimiento y Aseguramiento de Sistemas Informáticos 💻

Este es un proyecto web educativo diseñado para proporcionar información clara y estructurada sobre el mantenimiento de equipos tecnológicos, seguridad informática y procedimientos de operación.

## 📖 Descripción
La página web consta de una sección informativa principal y un formulario de contacto. Explica los diferentes tipos de mantenimiento (preventivo, correctivo y predictivo) y la importancia de asegurar la operación del sistema mediante respaldos y actualizaciones de software.

## 🛠️ Tecnologías Utilizadas
* **HTML5**: Estructura semántica del contenido.
* **CSS3**: Estilos personalizados y diseño adaptable.
* **Bootstrap 5.3.3**: Framework para diseño responsivo y componentes (como el menú desplegable).
* **JavaScript**: Funcionalidad de los componentes de Bootstrap.
